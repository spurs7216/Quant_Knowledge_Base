## Role of the chapter

This chapter provides the geometric objects that make convex optimization possible.

It is not only a definitions chapter. It gives the exact geometric statements that later become:

- feasible-set convexity
- separating certificates
- supporting hyperplanes for optimality
- cone duality for generalized inequalities and semidefinite structure

## Core mathematical objects

- affine combination
  $$\sum_{i=1}^k \theta_i x_i, \qquad \sum_{i=1}^k \theta_i = 1$$
- convex combination
  $$\sum_{i=1}^k \theta_i x_i, \qquad \theta_i \ge 0,\quad \sum_{i=1}^k \theta_i = 1$$

  $$x \preceq_K y \iff y-x \in K$$

  $$K^\ast = \set{y \mid x^\trans y \ge 0 \text{ for all } x \in K}$$
- supporting hyperplane at $x_0$
  $$a^\trans x \le a^\trans x_0 \quad \text{for all } x \in C$$

## Theorem and derivation spine

### Convex and affine sets

A set $C \subseteq \R^n$ is affine if
$$\theta x_1 + (1-\theta)x_2 \in C \quad \text{for all } x_1,x_2 \in C,\; \theta \in \R,$$
and convex if the same condition holds only for $0 \le \theta \le 1$.

The real durable point is that affine sets are closed under full lines, while convex sets are closed only under line segments. Equality constraints therefore create affine geometry; convex inequalities create convex feasible regions.

### Separating hyperplane theorem

For nonempty disjoint convex sets $C$ and $D$, there exist $a \ne 0$ and $b$ such that
$$a^\trans x \le b \quad \text{for all } x \in C, \qquad a^\trans x \ge b \quad \text{for all } x \in D.$$

In the special proof case emphasized in the book, one chooses closest points $c \in C$ and $d \in D$, then defines
$$a = d-c, \qquad b = \frac{\norm{d}_2^2-\norm{c}_2^2}{2}.$$
The resulting hyperplane bisects the segment between $c$ and $d$ and is orthogonal to it.

### Why separation matters

This theorem is the geometric ancestor of:

- dual lower bounds
- optimality certificates
- supporting hyperplane arguments for convex functions
- the theorem-of-alternatives style logic used all over optimization

### Supporting hyperplane theorem

If $C$ is convex and $x_0 \in \partial C$, then there exists a nonzero $a$ such that
$$a^\trans x \le a^\trans x_0 \quad \text{for all } x \in C.$$

This means every boundary point of a convex set admits a tangent hyperplane that contains the set in one closed halfspace. Later, when a sublevel set of a convex objective touches the feasible set at an optimum, this is exactly the geometry behind first-order optimality.

### Dual cones

The dual cone is defined by
$$K^\ast = \set{y \mid x^\trans y \ge 0 \text{ for all } x \in K}.$$

The chapter's durable examples are:

- $(\R_+^n)^\ast = \R_+^n$
- $(S_+^n)^\ast = S_+^n$ under the trace inner product
- for a norm cone $K=\set{(x,t)\mid \norm{x}\le t}$, the dual cone is the cone generated by the dual norm

The point is not just notation. Cone duality is the geometric reason linear, second-order cone, and semidefinite constraints all admit coherent dual problems.

### Generalized inequalities

With a proper cone $K$, the generalized inequality
$$x \preceq_K y \iff y-x \in K$$
lets us extend order ideas beyond scalar inequalities. But unlike the order on $\R$, generalized inequalities are usually only partial orders, so minimal and minimum elements no longer coincide automatically.

That distinction matters later in multicriteria optimization and Pareto geometry.

## Failure modes and misuse points

- treating convexity as a visual intuition instead of checking the actual closure properties
- forgetting that generalized inequalities are usually partial, not total, orders
- using the word "dual cone" formally without understanding the supporting-hyperplane interpretation
- importing cone notation into a problem where the chosen cone is not proper

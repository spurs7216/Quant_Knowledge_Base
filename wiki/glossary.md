# Glossary

This note collects short definitions for recurring terms in the vault. Expand entries here only when the concept does not yet deserve its own durable note.

## Alpha

Expected excess return or abnormal return relative to a benchmark, model, or neutral reference frame. Always state the benchmark and measurement context.

## Factor

A systematic return dimension, exposure, or ranking signal used for explanation, prediction, or portfolio construction. Distinguish risk factors from tradable factors when relevant.

## Identifier

A field used to refer to an entity across files or systems. Stability differs by vendor and data domain, so join rules matter as much as the label itself.

## Universe

The set of securities eligible for a study after all sampling and filtering rules. Universe definition is part of the claim, not metadata.

## Look-ahead Bias

Using information that would not have been available at the decision timestamp. This is a core vault lint target.

## Expansion Rules

- Promote an entry into its own note when it attracts repeated links or requires examples and caveats.
- Link to identifier, dataset, method, and risk notes when the short definition stops being enough.
